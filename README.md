# Résumé Shell


Résumé Shell is a shell looking website that you can use as your resume.


## Usage
Change the following vars in info.js file with your information:
```javascript
skillsInfo
experienceInfo
educationInfo
projectsInfo
aboutmeInfo
contactInfo
```
